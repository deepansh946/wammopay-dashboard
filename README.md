# Wammo Pay Dashboard

## Repo Setup

### MySQL Installation - https://dev.mysql.com/downloads/workbench/

- Frontend

  - `cd frontend`
  - `npm install`
  - `npm start`

- Backend
  - `cd backend`
  - `npm install`
  - `npm run dev`

Make sure you work in a new branch with your name as the name of branch. Happy Coding!
